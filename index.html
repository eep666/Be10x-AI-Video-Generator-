API Key Configuration Required

To use this application, please configure your Google Gemini API key in your Vercel project settings.

Instructions for Vercel:

  1. Go to your project's dashboard on Vercel.
  2. Navigate to the Settings tab.
  3. Click on Environment Variables in the side menu.
  4. Create a new variable with the name API_KEY.
  5. Paste your Google Gemini API key into the value field.
  6. Click Save and then redeploy your project for the changes to take effect.
